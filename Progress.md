# Fetures

## Lexical Analyser

**The `Lexical Analizer` supports these `Tokens` bellow:**

- **Delimiters:**
- `(`
- `)`
- `{`
- `}`
- `,`
- `;`
- `.`

- **Arithmetic Operators:**
- `+`
- `-`
- `*`
- `/`

- **Locgial Operators:**
- `!`
- `!=`
- `=`
- `==`
- `<`
- `<=`
- `>`
- `>=`

- **General:**
- `//`

- **If tokens does not match any of above, then that recognized as `Invalid Token`**
