# Fetures

**Lexical Analyser**
The *Lexical Analizer* supported these *Tokens* bellow:

    - Delimiters:
        - `(`
        - `)`
        - `{`
        - `}`
        - `,`
        - `;`
        - `.`

    - Arithmetic Operators: 
        - `+` 
        - `-` 
        - `*` 
        - `/` 
    
    - Logical Operators: 
        - `!`
        - `!=`
        - `=`
        - `==`
        - `<` 
        - `<=` 
        - `>`
        - `>=`

    - General:
        - `//`

    - If tokens does not match any of above, then that recognized as `Invalid Token`
